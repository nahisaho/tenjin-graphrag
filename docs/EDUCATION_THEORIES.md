## 🔗 リレーションシップタイプ

グラフデータベースで使用されている関係タイプの説明：

| タイプ | 説明 | 例 |
|--------|------|-----|
| `PROPOSED` | 理論家が理論を提唱 | ピアジェ → 構成主義 |
| `DEVELOPED` | 理論家が理論を発展 | メイヤー → マルチメディア学習理論 |
| `INFLUENCED` | 理論家間の影響関係 | ヴィゴツキー → バンデューラ |
| `EXTENDS` | 理論の拡張 | 社会的構成主義 → 構成主義 |
| `CONTRADICTS` | 理論間の対立 | 行動主義 ↔ 構成主義 |
| `COMPLEMENTS` | 相補的関係 | 認知負荷理論 ↔ マルチメディア学習理論 |
| `INCLUDES_CONCEPT` | 理論が概念を含む | 自己決定理論 → 自律性 |
| `RELATED_TO` | 理論間の関連 | 認知負荷理論 → 情報処理理論 |
| `THEORETICALLY_GROUNDED_IN` | 教授法の理論的基盤 | 足場かけ → 社会文化理論 |
| `SUPPORTS` | エビデンスが理論を支持 | メタ分析 → 認知負荷理論 |
| `CHALLENGES` | エビデンスが理論に挑戦 | 研究 → 学習スタイル理論 |
| `EFFECTIVE_FOR` | 文脈での有効性 | 認知負荷理論 → オンライン学習 |
| `APPLICABLE_IN` | 教授法の適用場面 | PBL → 医学教育 |

## 🛠 使用方法

### MCPツールでの検索

```python
# 理論検索
await search_theories(query="認知負荷", paradigm="cognitivism")

# 教授法推薦

```

### Cypherクエリ例

```cypher
// パラダイム別理論一覧
MATCH (t:Theory)
RETURN t.paradigm, collect(t.name) as theories
ORDER BY size(collect(t.name)) DESC

// 理論に関連するすべてのエンティティ
MATCH (t:Theory {id: "self-determination-theory"})-[r]-(related)
RETURN t, r, related

// 高エビデンスの教授法と理論的基盤
MATCH (m:Methodology {evidence_level: "high"})-[:THEORETICALLY_GROUNDED_IN]->(t:Theory)
RETURN m.name, collect(t.name) as theories
```
